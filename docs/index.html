This is a landing page for the hide-statement chrome extension.

It's a tiny chrome extension which hides the problem statement in a Codility challenge (thus making it even more challenging). Its functionality be toggled on/off by clicking on its icon (the blue or gray H) in the icon bar.

If you have chrome, you should install the extension.

An alternative to the chrome extension is to use a bookmarklet:

Drag (don't click!) this blue link to your bookmark bar: Hide Problem Then, once you have the codility problem loaded, click "Hide Problem" in your bookmark bar.

This is less convenient than the chrome extension, since you have to click it every time you reload or change the codility page, e.g. when you change languages.
